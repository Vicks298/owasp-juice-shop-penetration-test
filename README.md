# OWASP Juice Shop Web Application Penetration Test

## Overview

This repository contains a professional web application penetration testing assessment performed against the OWASP Juice Shop vulnerable application.

The assessment followed a structured penetration testing methodology, including:

- Reconnaissance
- Enumeration
- Vulnerability Assessment
- Authentication Testing
- Authorization Testing
- Exploitation
- Risk Assessment
- Professional Reporting

---

## Scope

Target:

OWASP Juice Shop

---

## Methodology

The assessment included:

- Manual Reconnaissance
- Endpoint Enumeration
- Authentication Testing
- Authorization Testing
- SQL Injection Testing
- Cross-Site Scripting (XSS) Testing
- Information Disclosure Testing
- Insecure Direct Object Reference (IDOR) Testing

---

## Key Findings

- SQL Injection Authentication Bypass (Critical)
- Broken Access Control (IDOR)
- Information Disclosure via API
- Unauthorized Basket Modification
- Unauthorized Basket Deletion
- Stored Cross-Site Scripting (XSS)
- Authentication & JWT Assessment
